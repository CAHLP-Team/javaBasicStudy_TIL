### 배열과 그룹핑
배열은 연관된 정보를 그룹핑 할 때 사용하는 데이터 타입.

많은 데이터를 묶어서 보관하기 좋게 하기위해서 배열을 사용한다.

예를 들어서 학급을 나눌때 사람이 많으면 학년과 반을 나누어 보다 효율적으로 호명이 가능해진다.

배열은 여러 개의 데이터를 저장 가능

데이터 타입 [] 변수명 하면 배열이 생성된다

String[] a = {"a","b","c"};
System.out.println(a[0]); //a
System.out.println(a[1]); //b
System.out.println(a[2]); //c

배열의 시작은 1이 아닌 0!!!!

     return;
}

만약 메소드가 없다면?

### 입력값
매개변수와 인자

코드 읽는 요령
메소드를 읽으려면 메소드를 어떻게 사용하고 입력값과 출력값이 무엇인지 보고 메소드의 정의를 보는게 더 도움이 된다.

return 돌려주다

main 앞에 void 는 빈 공간

메소드 앞에 void를 쓰면 빈 공간을 선언하는 것이다.
